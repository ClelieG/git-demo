# README FOR git-demo

Working with git 

    -workflow of git 
        1. "git init" once to initialize 
        the repo
        2. write some code 
        3. "git add ." / "git commit -m" 
        "some message" whenever you want to make a commit 
        4. "git push origin main" push the changes to github 
        5. repeat steps 2-4
